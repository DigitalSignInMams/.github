# Project Description

The current sign-in system at Mass Academy is very mechanical and time-consuming. This project aims to improve the efficiency and technology related to the sign-in and sign-out methods. This sign-in system will provide students with an automated technology to record the sign-in and sign-out times for students. The data will be stored on a hard drive and uploaded to the cloud for admins to access. This sign-in system will also provide admins with a simplified method to handle attendance data and access past data.

# Target Audience

Students to make the sign-in process quicker and more accurate
School administrators to make the process of transferring the sign-in records to the academy’s attendance system, and allowing for the records to be easily viewable online.

# Project Requirements
- This technology will be able to store daily user sign-in and sign-out data digitally.
- This technology will be able to show Ms. Julie and the administration sign-in and sign-out reports in a user-friendly way.
- This digital technology must reduce the amount of time it takes to sign in using the traditional “paper and pen” method.
